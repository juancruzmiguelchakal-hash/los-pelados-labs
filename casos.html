KuchaBicho - Control de Plagas

Web Formularios Precios Dinámicos

Desafío

El cliente necesitaba una web profesional que mostrara sus servicios de control de plagas con una tabla de precios interactiva y formularios de contacto integrados.

Solución

Desarrollé una web moderna con diseño oscuro, tabla de precios dinámica con filtros, integración de WhatsApp y formularios con FormSubmit.

Resultado

Web funcional entregada en 2 semanas. El cliente reportó aumento en consultas y mayor profesionalismo percibido por sus clientes.

VisionCraft - Portfolio Creativo

Portfolio Animaciones UI/UX

Desafío

Crear un portfolio visual que destacara entre la competencia con animaciones suaves y diseño premium.

Solución

Implementé AOS para animaciones, glassmorphism y una paleta de colores vibrante con un layout limpio y profesional.

Resultado

Portfolio moderno con excelente rendimiento y 100% responsive. Diseño que transmite profesionalismo y creatividad.
